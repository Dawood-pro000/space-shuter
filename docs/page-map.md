# Space Shutter - Page Map

## Public Pages
1. **Home** (`/`) - Hero, Features, Testimonials.
2. **Discover** (`/discover`) - Public feed showing the latest 10 discoveries.
3. **Pricing** (`/pricing`) - Subscription tiers (Free, Student, Pro).
4. **About Us** (`/about`) - Mission and vision.
5. **Contact Us** (`/contact`) - Contact form submission.
6. **Privacy Policy** (`/privacy`) - Legal text.

## Authentication
1. **Register** (`/register`) - Account creation.
2. **Login** (`/login`) - Sign in.
3. **Reset Password** (`/reset-password`) - Password recovery.

## User Dashboard (Protected)
1. **Space Feed** (`/feed`) - The primary feed for posts, articles, and NASA discoveries.
2. **Study Planner** (`/study-planner`) - Tool to generate and view learning paths.
3. **Journey Tracker** (`/journey`) - Visual progress map and achievements.
4. **Library** (`/library`) - Searchable repository of books and PDFs.
5. **Saved Content** (`/saved`) - Bookmarks for articles, books, and plans.
6. **Profile** (`/profile`) - User details, study hours, achievements.
7. **Settings** (`/settings`) - Account preferences and subscription management.
8. **AI Assistant** (`/assistant`) - Chatbot interface.

## Admin Dashboard (Protected, Admin Only)
1. **Overview** (`/admin`) - System health, analytics, top-level stats.
2. **Posts Management** (`/admin/posts`) - CRUD for articles (Draft, Scheduled, Published).
3. **Users Management** (`/admin/users`) - View, suspend, upgrade roles.
4. **Subscriptions** (`/admin/subscriptions`) - Track active plans and revenue.
5. **Books Management** (`/admin/books`) - Upload and manage library content.
6. **Categories** (`/admin/categories`) - Create and manage tags.
7. **Contact Requests** (`/admin/contacts`) - Review form submissions.
8. **Activity Logs** (`/admin/logs`) - Viewer for admin actions.
9. **Settings / Feature Flags** (`/admin/settings`) - Toggles for various site features.
10. **AI Control** (`/admin/ai`) - Trigger and monitor background AI jobs.
